Get Mukurtu

Mukurtu started as a one-off, stand alone, browser-based digital archive built specifically around the needs and cultural protocols of the Warumungu people in Tennant Creek, N.T., Australia. Since installing it in 2007 we have had numerous requests from Indigenous communities, cultural centers, libraries and archives for Mukurtu. With a National Endowment for the Humanities grant in 2010, we began production of the beta version of Mukurtu.

What's new with Mukurtu?

We plan to have a 1.0 release in Summer 2012. This gives us some time to get it right, and we're depending on your contributions, passion, and drive to help us get there.

How you can get involved

Feedback

We've come a long way, but we're really just beginning. As a non-profit group making free and open source software, we depend on you, the user, to tell us what you want, what went wrong, and help us to make it right.

We are open to hearing your interests and ideas on the best way to engage you. For now, you can send feedback about the platform with this online form, and contact us anytime by email, beta@mukurtu.org.

Community Collaboration

So far, the Mukurtu community consists of many people from all around the world, each with an important role in relation to the project. This includes local communities and community members, librarians, institutional stakeholders, open source software developers, supporters and other advocates.

In order to help the Mukurtu suit the needs of the groups and communities that want to use the software, we will be creating opportunities for community feedback and dialogue. These opportunties will be announced through the mailing list, and soon through social media.

Version History

Mukurtu 0.5 Demo Release Fall 2011

Environment

  • Full service content management system. Can be run on a production server, or hosted service, and locally on a laptop for offline and field access.
  • Collaborative environment with preconfigured, easy to use themes and templates.
  • Powered by Drupal 7.8, the open source platform adopted by 228 countries worldwide.

Platform

  • One click install. Install and be up and running in quickly.
  • Easily updated to regularly released builds with new features, bug fixes and optimizations.
  • Mukurtu/Drupal integration means standards-based support for all major web browsers on Linux, Windows, and Mac OS X, in addition to modern mobile frameworks (Android, iPhone)

Features

  • Cultural Protocols linked to content (items, collections) providing granular access levels to content in the archive.
  • Multiple licensing options: including specific traditional knowledge licenses for individual content or collections.
  • Powerful users, groups, and roles management. Use preconfigured compositions (library, museum, tribe) or easily modify to suit your needs.
  • Batch import media. Add pictures, movies, PDFs, Office documents and associated metadata in bulk.
  • Module Manager. Enable or disable a variety of frontend features and backend functions, such as image previews and advanced user administration.
  • Easy-to-use dashboard controls and Panel views to guide administrators and users.
  • Backup locally and offsite, automatically.

Environment

Platform

  • All supported (XP, OS X, etc). Note, CD Burning from the browser requires OS X and certain file permissions. The archive runs locally on an iMac in a MAMP package (Mac OSX, Apache, MySQL, PHP), or on a Windows PC running XAMPP. The package will also run in core Web server environments.

  • One click print from display page.
  • Single and batch upload.

Mukurtu CMS

Mukurtu will run on all operating systems through an easy-to-use, one-click installation process.

  • Windows
  • Mac OSX
  • Linux

System Requirements

LAMP or equivalent Mac OS or Windows stack (Linux, Apache, MySQL, PHP). See Drupal's requirements page.

  • Current Linux, Mac or Windows OS environment
  • Apache 2.x or Microsoft IIS 5+
  • MySQL 5.x, MariaDB 5.x, or PostgreSQL 8.x
  • PHP 5.2.5 or above, at least 32MB memory
    • PHP GD Library
    • ImageMagick
